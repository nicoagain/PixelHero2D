<h1>Pixel Hero 2D</h1>

<h3>
The meaning of this prototype is to learn for example how to switch between normal mode and ball mode and how the attacks changes in those modes, how to unlock upgrades like double jump, dash, ball mode and bombs. 
Also you'll learn about camera adjustment and some transitions between animations like fading when you collect an item, how to instantiate arrows and bombs, particles, etc.
</h3>
